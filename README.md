# 🎬 Generative Media Scripting Pipeline

An automated multimedia marketing script and AI asset generation pipeline powered by Open Source AI models and Streamlit. This application uses the **Llama-3.3-70B-Instruct** model via Hugging Face for copywriting and **FLUX.1-schnell** for high-quality scene rendering.

---

## 🚀 Features

- **Automated Script Writing:** Generates high-converting short-form marketing scripts using the AIDA framework based on your product name and target audience.
- **Visual Storyboard Planner:** Explodes the script into detailed visual scenes, camera movements, and optimized AI image prompts.
- **Real-Time Image Rendering:** Uses the advanced FLUX model to bring visual storyboard prompts to life instantly.
- **Voiceover Generation (TTS):** Converts the final script text into natural-sounding audio voiceovers using Google TTS (supports English, Roman Urdu, Urdu, and Hindi).
- **Downloadable Assets:** One-click download options for generated images, audio tracks, and the complete text production brief.
